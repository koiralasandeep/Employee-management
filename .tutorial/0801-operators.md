# 8-1. Using Operators with Strings

## a. + or concatenation operator
```python
# concatenate multiple strings to create one string
string1 = "Hello"
string2 = "World"
new_string = string1 + string2
print(new_string)
```

## b. * or repetition operator
```python
# Repeat a string 5 times
string = "tick tock "
new_string = string * 5
print(new_string)
```

<details>
  <summary>
    ðŸ’¡ Remember:
  </summary>
